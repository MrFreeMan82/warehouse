//          SERVER
1. Создать в БД таблицу либо таблицы которые представляют сущность.
2. Создать на сервере в пакете entity новую сущность
3. Настроим для нее аннотации соответствующие ограничениям в БД.
4. Создать mapping на сущность
5. Создать DAO для сущности, определить методы Создать, Изменить, Удалить, которые объявим в интерфейса в п.7.

//          CLIENT
6. Создать на клиенте в пакете proxy доверенного на стороне клиента для осущности.
7. В файле AppRequestFactory.java нужно создать интерфейс для работу с сущностью либо с некоторыми сущностями на сервере
    здесь для каждого запроса нкжно создать метод.

8. В пакете i18n определить нужные константы для представления сущности.
9. Создадим макет
10. Создадим представление.
11. Определим валидаторы если необходимы.
12. Определим события где нужно.
